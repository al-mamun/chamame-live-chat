Simple chat application for customer support.

== Description ==

= Plugin features =
* Working with standalone. Not depend on other services.
* Easy to use.
* Promising ;)

= Future enhancements =
* Custom label text.
* Custom look and feel.
* Operator nickname to display in chat console.
* Save and view chat history.
* Disable chat when operator is away.
* Alternative contact form for when operator is away.
* Notification email when visitor starts chat.

= Limitations =
* Multisite is not supported.

== Installation ==

1. Download plugin and unzip.
1. Upload `chamame-live-chat` to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
